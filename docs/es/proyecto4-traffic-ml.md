# Proyecto 4 - Clasificacion de Tráfico de Red con Machine Learning (v0.1)

## Objetivo
Crear un modelo de IA que clasifique tráfico de red como normal o malicioso.

## Importancia
Base de los sistemas de Detección de Intrusiones (IDS).

## Dataset
- UNSW-NB15
- CICIDS2017

## Metodología (borrador)
1. Preprocesamiento de tráfico
2. Selección de variables
3. Entrenamiento de modelos
4. Evaluación 
5. Exportación del modelo

## Resultado esperado
Modelo funcional de detección de intrusiones. 

## Caso de uso profesional
SOC, Blue Team, IA aplicada a defensa.
