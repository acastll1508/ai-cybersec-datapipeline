# Proyecto 2 - Detección de URLs Maliciosas con Machine Learning (v0.1)

## Objetivo
Desarrollar un modelo de Machine Learning que clasifique URLs como maliciosas o benignas.

## Importancia
La detección automatizada de sitios maliciosos es clave en inteligencia de amenazas y ciberseguridad ofensiva.

## Dataset
Dataset público de URLs maliciosas y benignas.

## Metodología (borrador)
1. Carga del dataset
2. Limpieza
3. Extracción de características
4. Entrenamiento del modelo.
5. Evaluación

## Resultado Esperado
Modelo funcional de clasificación de URLs.

## Caso de Uso Profesional
Aplicable en análisis de malware, phishing y detección temprana de amenazas.
