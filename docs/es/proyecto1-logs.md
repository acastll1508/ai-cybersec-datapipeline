# Proyecto 1 - Análisis y Visualización de Logs de Ciberataques (v0.1)

## Objetivo 
Analizar logs reales de ataques informáticos para identificar patrones de comportamiento malicioso 
mediante Python y técnicas de análisis de datos.

## Importancia
Este proyecto representa una de la tareas mas comunes en un SOC: analizar registros de seguridad 
para detectar ataque activos. 

## Dataset
- Logs de Honeypots SSH (fuerza bruta)
- Logs de firewall
- Logs de servidores web Apache

## Metodología (borrador)
1. Carga de datos
2. Limpieza de registros
3. Identificación de IPs maliciosas
4. Análisis de frecuencias
5. Visualización de ataques

## Resultado Esperado 
- Identificación de IPs mas activas
- Horarios pico de ataque
- Gráficos de actividad

## Caso de Uso Profesional
Aplicable en Centros de Operaciones de Seguridad (SOC)
