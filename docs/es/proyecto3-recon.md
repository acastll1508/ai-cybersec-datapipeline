# Proyecto 3 - Automatización de Reconocimiento Ofensivo (v0.1)

## Objetivo
Automatizar el proceso inicial de reconocimiento en hacking ético usando Python.

## Importancia
El reconocimiento es una de las fases mas importantes en Ciberseguridad ofensiva.

## Metodología (borrador)
1. Recolección de headers HTTP
2. Escaneo de puertos
3. Detección de tecnologías
4. Generación de reportes

## Resultado esperado
Herramienta automática de reconocimiento ofensivo.

## Caso de uso profesional
Soporte para Red Team y Pentesting.
